# UFO Shooting Game

A simple browser shooting game built with **HTML5 Canvas**, **CSS**, and **JavaScript**.  
The player controls a spaceship to shoot down incoming UFOs and survive as long as possible.

## Project Overview

This project is a 2D shooting game developed using the HTML5 Canvas API.  
The main goal of the game is to control the aircraft, avoid enemy attacks, and destroy UFOs to earn points.

The project was built to practice:

- JavaScript game logic
- HTML5 Canvas rendering
- Collision detection
- Keyboard event handling
- Real-time animation with `requestAnimationFrame`

## Features

- Player-controlled spaceship movement
- Bullet shooting system
- Enemy UFO generation
- Collision detection between bullets and enemies
- Score tracking
- Game over mechanism
- Simple and responsive game interface

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **HTML5 Canvas**

## How to Play

- Use the **Arrow Keys** or **A / D** keys to move the spaceship
- Press **Space** to shoot
- Destroy UFOs to gain points
- Avoid getting hit by enemies
- The game ends when the player loses all health or is hit
